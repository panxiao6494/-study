自增自减详解
案例要求： 判断前置和后置时表达式的值和变量的值
思考重点：
1.一定区分表达式和变量，num++,++num这些都是表达式，有一个返回值，不是变量的值，num才是变量
2. num++, ++num的作用都是一样 就是将变量 num的值+1
3. num++, ++num不一样在于返回值不一样
3.1. num++是先使用num作为返回值，然后再对变量num加1，他的返回值是1， num的值是2
3.2. ++num是先对num加1，然后再使用num的值作为返回值，这是返回值为2， num的值也为2
var num =1; // console.log(num++, ++num, num); //1,3,3 var nNum1 = num++ + ++num + num; //7
判断闰年案例
案例要求： 用户输入年份，输出是否是闰年
思维过程：
1. 关键判断什么是闰年，这就是算法。闰年就是能被4整除并且不能被100整除，或者能被400整除的就是闰年，否则就是平年
2. 2004是闰年，1999不是闰年； 2000年是闰年，1900不是闰年。闰年366天，2月有29号
3. 输入语句是prompt,输出语句是alert
4. 定义一个变量接受用户输入的值，最好进行一步类型转换，然后就是利用if...else分支结构，写好条件语句
var sYear = prompt('请输入想要查询的年份'); var nYear = parseInt(sYear); if( ((nYear%4 == 0) && (nYear%100 != 0)) || nYear%400 ==0 ){ alert("我是闰年"); }else { alert("我不是闰年") }
数字补零案例
案例要求： 用户输入一个数，如果是一位数则在前面添加2个零，如果是两位数则在前面添加1个零，如果是三位数则什么也不做
思维过程：
1. 定义一个变量就收用户输入的值
2. 主要的逻辑在于对于输入的数字进行判断，小于10的一位数，大于等于10且小于100的叫二位数，剩下的不处理
var nNum = prompt("请输入一个数字："); if(nNum<10){ nNum = '00' + nNum.toString(); }else if(nNum>=10 && nNum<99){ nNum = '0' + nNum.toString(); }else { nNum = nNum.toString(); } alert(nNum);
